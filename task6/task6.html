Use the Dow Jones index (data set dowjones) to do the following: (a) Produce a time plot of the series. (b) Produce forecasts using the drift method and plot them. (c) Show that the graphed forecasts are identical to extending the line drawn between the first and last observations. (d) Try some of the other benchmark functions to forecast the same data set. Which do you think is best? Why?

a)Produce a time plot of the series.

Duomenyse matomas trendas, nėra sezoniškumo:

data=dowjones
plot(data)                                                # a) dalis, išbrėžiami duomenys

b) Produce forecasts using the drift method and plot them.

x=rwf(data, drift=TRUE, h=30)                             # b) dalis, draft prognozė 
plot(x)

Atsižvelgia į trendą. trendas su random walk.

(c) Show that the graphed forecasts are identical to extending the line drawn between the first and last observations

plot(x)                                                   # c) dalis, įrodome, kad sujungus pirmą ir
lines(c(0,78),c(data[1],data[78]),lwd= 2, col="BLUE")     #    paskutinį tašką, gauname tokį patį rezultatą

d)Try some of the other benchmark functions to forecast the same data set.

vidurkio prognozė:

y=meanf(data, h=30)                                       
plot(y)

Naive prognozė:

y=naive(data, h=30)                                       
plot(y)                                                       

tsdisplay(data, plot.type="scatter")

Vidurkio prognozė yra prastesnė, nes akivaizdus trūkio taškas. Tiktų naive arba drift metodas. Kadangi paskutiniai duomenys pradėjo ir leistis, galime tik spėti ar kiti duomenys kris ar kils. Todėl rinkčiausi naive modelį, nes prognozė nei kyla nei leidžiasi, tinka šiame neužtikrintume dėl ateities.
