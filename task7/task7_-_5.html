((c) Try various benchmark methods to forecast the training set and compare the results on the test set. Which method did best?

Naudosime paprasiausius modelius: mean, naive, snaive ir drift:

par(mfrow=c(2,2))
vid <- meanf(data1, h=20)               # prognoz, lygi duomen vidurkiui
plot(vid)

naive <- naive(data1, h=20)             # prognoz, kuri atsivelgia tik  paskutin duomen reikm
plot(naive)

snaive <- snaive(data1, h=20)           # preitas ufiksuotas sezonas yra naudojamas prognozuoti ateit
plot(snaive)

drift <- rwf(data1, drift=TRUE, h=20)   # atsivelgia  paskutin duomen reikm ir prideda vidutin svyravim
plot(drift)                             # vidutinis svyravimas yra maai nutols nuo 0, taigi beveik sutampa su naive prognoze

((d) For the best method, compute the residuals and plot them. What do the plots tell you?

iuo atveju reikia metodo ilgam laikotarpiui, tad pasirinksiu drift metod kaip geriausi. Kaip matome yra dsningumas tarp paklaid. Jos koreliuoja ir nra baltas triukmas, taigi nra pats geriausias modelis, taiau i vertint - geriausias prognozuojant naujausius duomenis:

naive= naive(hsales)
res <- residuals(naive)
Acf(res)                 # paklaidos yra koreliuotos

plot(res)                # matosi ioks toks dsningumas, nra baltas triukmas

hist(res, breaks="FD")   # atrodo kaip normalusis skirtstinys

shapiro.test(res)        # atmetame H0 hipotez, jog paklaidos normaliosios. Paklaidos neturi normaliojo skirstinio
## 
##  Shapiro-Wilk normality test
## 
## data:  res
